# rpa-shops

A dynamic shop system with pedagogical spawning and target integration.

## Features
- **Config Driven**: Add shops via `config.lua` (coords, ped model, items).
- **Ped Spawning**: Automatically handles shopkeeper peds.
- **UI**: Clean Grid UI for purchasing items.

## Installation
1. Ensure `rpa-lib` is started.
2. Add `ensure rpa-shops` to your `server.cfg`.

## Configuration
Edit `config.lua` to add new shop locations and inventories.
